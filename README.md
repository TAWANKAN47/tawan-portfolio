## Getting Started

To run the project locally:

# 1. Clone the repository
git clone https://github.com/your-username/tawan-portfolio.git

# 2. Navigate to the project directory
cd tawan-portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
